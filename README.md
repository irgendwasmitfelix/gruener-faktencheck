Unabhängige Artikel und Links zur Grünen Partei Deutschland.

⚠️ **Das ist kein offizielles Projekt der Grünen!**

## Features
- 📄 Übersichtliche Sammlung von Faktencheck-Artikeln
- 🔍 Suchfunktion nach Artikeln
- 🌓 Dark Mode Support
- 📱 Responsive Design
- 🚀 Optimiert für SEO (Sitemap, Schema.org, JSON-LD)
- ⚡ Statische HTML-Generierung für bessere Performance

## Kategorien
- **Wirtschaft** - Artikel zu Wirtschaftspolitik
- **Innenpolitik** - Innenpolitische Analysen
- **Außenpolitik** - Außenpolitische Themen

## Technologie
- React 18
- Vite (Build-Tool)
- React Router DOM (Navigation)
- React Helmet (SEO Meta-Tags)
